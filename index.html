HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • You can change the font size in each section.
  • HTML is used to create web pages.
  • We can use semantic HTML elements to indicate the role of their content.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When a developer wants to style an element directly in the HTML file, they use inline CSS.
  • When using an external style sheet, we link the file using a "link" element placed in the HTML's "head".
  • A selector defines the element or attributes to which the rules, or declarations, will apply.

Git

  • Git is used to let multiple people work on the same project without getting in eachothers way.
  • You need to submit a pull request in order to update the code.
  • It is always beneficial to use a feature instead of being in the main.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • It is beneficial to check what directory you are in using git status.

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites.
  • Using JavaScript, you can power your websites with effects, interactivity, data communication, and much more.
  • Every time you do something with a webpage, such as click a button, retrieve data, or fill out a form, JavaScript is at work.
  • A variable is simply a named container that allows us to store data in our code.
  • whereas the style.css file is linked within the "head" element, the script.js file is linked within the "body" element.
  • Control flow is the order in which a computer executes code in a script.
  • When the computer encounters an if statement in a JavaScript file, it will check to see if the condition that we provide is truthy.
